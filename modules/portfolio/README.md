# Portfolio module

Adds a browsable project gallery to the site. The index route (`/portfolio`) renders a grid of case-study cards filterable by room type and design style, with a custom hover cursor on desktop. Each project links to its own detail page (`/portfolio/[slug]`) which includes a hero image, a three-column metadata band (room, brief, design call), a long-form case-study story in Sanity Portable Text with a sticky TOC sidebar, a justified photo gallery with lightbox, optional before/after comparison sliders, related testimonial and services, and a contextual sticky CTA chip. A dedicated before/after page (`/portfolio/before-after`) aggregates all projects that have slider pairs into a single scrollable reveal page. The module introduces two Studio schemas: `portfolioPage` (singleton controlling the index hero copy) and `project` (orderable collection of case studies, drag-to-reorder via the orderable-document-list plugin). Core dependencies that are intentionally kept outside the module folder: `BeforeAfterSlider.tsx`, `CaseStudyTOC.tsx`, `StickyCTAChip.tsx`, `SanityImage.astro`, `SectionHeading.astro`, `FinalCta.astro`, and the `getSectionVisibility` helper.
