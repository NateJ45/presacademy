# Sermons module

Adds a sermons/media section. The index route (`/sermons`) shows a featured latest message, an archive grid (newest first, grouped by series/speaker via the card meta), and a persistent "Watch Live" link. It degrades gracefully: when no sermons exist in Sanity it shows a "watch online" state pointing at the church livestream, so the page is never empty. Each sermon links to `/sermons/[slug]` with an embedded YouTube/Vimeo player, date/speaker/series/scripture meta, optional audio link, and Portable Text notes. Two Studio schemas: `sermonsPage` (singleton: hero + livestream URL) and `sermon` (collection). Core dependencies kept outside the module: `Hero.astro`, `SectionHeading.astro`, `FinalCta.astro`, `SanityImage.astro`, `PortableText.tsx`, and `breadcrumbSchema`. See `docs/modules/sermons.md` for the enable steps.
