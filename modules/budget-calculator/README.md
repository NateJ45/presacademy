# Budget Calculator module

Adds an interactive project budget estimator at `/calculator`. Visitors pick a room type from a dropdown, choose a scope level from a radio group, optionally check add-ons, and immediately see a live estimate range. The estimate always shows without requiring an email; an optional "Email me this estimate" form is available but never gates the result. Calculator logic lives in the `BudgetCalculator` React island, which hydrates with `client:visible`. The Studio schema (`budgetCalculator` singleton) controls all room types, base cost ranges, scope options, add-on labels and cost ranges, result copy, and the CTA label, all editable without touching code. A coming-soon state renders when no rooms are configured, so the route never crashes on an underconfigured doc. Core dependencies that are intentionally kept outside the module folder: `SectionHeading.astro`, `sectionVisibility` (from `@/lib/sectionVisibility`), `subscribeEmail` (from `@/lib/subscribe`), and all shadcn UI primitives.
