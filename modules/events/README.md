# Events module

Adds a church/organization events system. The index route (`/events`) lists upcoming one-time events and the weekly/recurring rhythms of the organization, with graceful empty states and a static fallback list of recurring rhythms so the page is complete before Sanity is connected. Each event links to its own detail page (`/events/[slug]`) with schedule, location, an optional image, a Portable Text description, and an optional registration link. The module introduces two Studio schemas: `eventsPage` (singleton controlling the index hero) and `event` (a collection of recurring and one-time events). Core dependencies kept outside the module: `Hero.astro`, `SectionHeading.astro`, `FinalCta.astro`, `SanityImage.astro`, `PortableText.tsx`, and the `breadcrumbSchema` helper. See `docs/modules/events.md` for the enable steps.
