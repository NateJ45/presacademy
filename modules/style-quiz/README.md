# Style Quiz module

Adds an interactive style-preference quiz at `/quiz`. Visitors answer four image-choice questions, optionally answer two qualifier questions (timeline, budget), and receive a design-archetype result with a description and a contextual CTA. The result can optionally be gated behind an email form (mode is configurable in Studio). Quiz logic lives in the `StyleQuiz` React island, which hydrates with `client:visible` to keep TTI low. The Studio schema (`styleQuiz` singleton) controls all copy, questions, archetype descriptions, email-gate mode, and routing rules without any code changes. A coming-soon state renders when fewer than 2 questions or 2 archetypes are configured, so the route never crashes on an underconfigured doc. Core dependencies that are intentionally kept outside the module folder: `SectionHeading.astro`, `sectionVisibility` (from `@/lib/sectionVisibility`), `subscribeEmail` (from `@/lib/subscribe`), and all shadcn UI primitives.
